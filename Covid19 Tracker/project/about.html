About

Covid-19 tracker is a continuously updated source of COVID-19 data. We collect and analyze the best data available on cases, deaths and tests to help the public, policymakers, and healthcare professionals worldwide respond to the pandemic.

This tracker is developed using a real time api which provides updated data. The platform is vital for information consolidation, and all gathered information is open to use for the public.

We aim to use technology and make information more accessible, gather talented people all around the world to win the fight!
